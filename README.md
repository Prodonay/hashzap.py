# HashZap - Chat Simples com Flet

Este projeto é uma aplicação simples de chat desenvolvida em Python utilizando o framework **Flet**, com foco em aprendizado de interfaces gráficas e eventos interativos.

## 📌 Objetivo

O objetivo do projeto é praticar:

* Criação de interfaces gráficas com Flet
* Manipulação de eventos (cliques e inputs)
* Entrada e exibição de dados em tempo real na interface
* Estruturação de aplicações Python com funções e escopo

## 🚀 Funcionalidades

* Tela inicial com botão para iniciar o chat
* Popup para inserção do nome do usuário
* Exibição de mensagem de entrada no chat
* Campo para envio de mensagens
* Exibição das mensagens enviadas na tela

## 🧠 Como funciona

1. O usuário clica em **"Iniciar Chat"**
2. Insere seu nome no popup
3. Entra no chat
4. Pode enviar mensagens que são exibidas na tela com o formato:

```
Nome: mensagem
```

## 🛠️ Tecnologias utilizadas

* Python 3
* Flet (UI framework)

## ▶️ Como executar

1. Instale a dependência:

```bash

```
